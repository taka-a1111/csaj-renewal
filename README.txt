デザインカンプ v3
================================================

v1への11項目のご指摘＋取り扱いコード関連ページの追加を
全て反映した本番マルチページ構成のサイトです。

【ファイル構成（全32ページ）】
■ メイン6ページ
   index.html              TOP
   company.html            会社概要
   business.html           事業概要
   code.html               取り扱いコード一覧（全25コード）
   theme.html              最近のテーマ
   news.html               お知らせ
■ 取り扱いコード詳細 25ページ
   code-retran.html ほか24コード（各コード個別ページ）
■ 参考1ページ
   index-color-alt.html    別配色案（ロイヤルブルー＋ミント）

【v1→v3 主要変更点】

１．PC/SPツールバーを削除し、本番マルチページサイトとして
   構成（公開時PC/SP自動判別）
２．全ページでスクロール追従ヘッダーを実装
３．ヒーローの本文を新しい文章に差し替え
４．「事業概要を見る」ボタンの色を「お問い合わせ」と
   同色に変更、矢印を削除
５．ヒーローの設立年・コード数・テーマ数 3stats を削除
６．ヒーロー右側に解析メッシュ風のSVGビジュアルを追加
   （余白対応）
７．事業の3本柱の文章を全文反映
   ・「過渡時熱水力解析」→「熱水力解析」
   ・シビアアクシデント解析の文章を新文章に
   ・PRAサブタイトルを「Probabilistic Risk Analysis」に
   ・PRA説明文に「通常運転時や停止時のほか」を追加
８．3本柱のアイコンを差し替え
   ・熱水力：青→紫のグラデーション波形
   ・シビアアクシデント：炉心溶融を抽象化したイメージ
   ・PRA：矩形波（リスクモニター風）
９．本文フォントを 15px → 16px に拡大
   グレー文字（ink-2, ink-3）を濃いトーンに調整
１０．TOP「最近のテーマ」を各分野4件に絞り、「詳しく見る→」
   は中身を登録したテーマにのみ表示
１１．参考：白青基調を残したロイヤルブルー＋ミント系の
   別配色案を1パターン用意（index-color-alt.html）

【セキュリティ対策（機微情報配慮）】

【ご確認方法】
   zipを展開後、index.html をブラウザでダブルクリックで
   ご覧いただけます。ページ上部のナビメニューや「コード一覧」
   からの遷移、コード詳細ページ間の前後リンクなど、本番と同じ
   挙動でご確認いただけます。
